--[ Для сталкера ]--
monsters_friendly : дружит ли сталкер с мутантами, (true/false).
stalkers_friendly : дружит ли сталкер с другими сталкерами, (true/false).
corpse_money : сумма электронных денег НПС, которую ГГ может извлечь из трупа при наличии утилит для взлома.
avoid_mines : иммунитет к SGM минам, (true/false).
avail_release : брать ли труп под контроль "уборщика трупов", (true/false).
combat_exit_delay : держать ли паузу после боя, (true/false).
can_be_wounded : может ли сталкер быть раненным, (true/false).
corpse_detection : может ли сталкер мародерствовать, (true/false).
gather_detection : будет ли сталкер искать себе более мощное оружие, чем у него сейчас (true/false).
wounded_detection : будет ли сталкер оказывать помощь раненым (true/false).

--[ Для мутанта ]--
stalkers_friendly : дружит ли мутант со сталкерами, (true/false).
monsters_friendly : дружит ли мутант с другими мутантами, (true/false).
capture_confirmed : чувствительность к препарату X8П25, (true/false).
aura_inactivity_active : активация ауры бездействия ГГ, (true/false).
aura_inactivity_radius : радиус воздействия ауры (значение в метрах).
aura_inactivity_frequency :  частота с которой обновляется аура. Стандартное значение 1000, это одна секунда реального времени.
aura_inactivity_effector : визуальное сопровождение действий ауры, (true/false).
aura_inactivity_disable_info : инфопоршень, наличие которого отлючает ауру.
particle_name : путь к партиклу.
particle_bone : кость воздействия партикла.
particle_sound : звук сопровождающий партикл.

--[ Для ящика-инвентаря ]--
inventory_use_info : выдача инфопоршня при использовании инвентаря.

--[ Для брони ]--
hud_carrying_outfit : установка указанного в значении худа при надетом бронекостюме или шлеме.
has_antigas : наличие противогаза в шлеме или бронекостюме, которое сопровождается звуком дыхания ГГ, (true/false).

--[ Для оружия ]--
increasing_skill_info : инфопоршень, наличие которого дают бонус increasing_skill_point к урону.
increasing_skill_point : бонус к урону.

--[ Для детектора ]--
radar_area : радиус идентификации артефактов на КПК (значение в метрах).
radar_accuracy : точность идентификации артефактов. Если true - на метке будет указано точное название артефакта, а false - просто надпись "артефакт" на метке.

--[ Для предмета который можно использовать ]--
on_use_info : выдача инфопортации (значение параметра) после использования предмета.
activation_time : время активации предмета, в секундах. Пока предмет в стадии активации, ГГ прячет своё оружие в кобуру.
reception_money : выдача или потеря денег (значение параметра) после использования предмета.
after_used_transform : спаун этого объекта (значение параметра) сразу после использования предмета.
eat_toxicity : повышение или понижение уровня токсичности организма ГГ, (значение от -1.0 до 1.0).
eat_psy_health : повышение или понижение состояния пси-здоровья ГГ, (значение от -1.0 до 1.0).

--[ Для бинокля, ножа и активаторов ]--
activate_binocular : название секции для которой используется активатор бинокля.
activate_knife : название секции для которой используется активатор ножа.
activate_section : принадлежность к активатору.
